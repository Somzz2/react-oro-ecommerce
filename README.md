# E-COMMERCE DE ANÉIS/JÓIAS

Projeto desenvolvido utilizando REACT CRA, Typescript, React Routes, CSS styled-componentes com o intuito de desenvolver o front-end de um e-commerce com carrinho de compras.


### Instalação

- `npm install`

### Para rodar

- `npm start`
